## Approval Workflow for SharePoint File External Sharing
This Power Automate flow enables admin-approved external file sharing. It is especially useful for public sharing scenarios, such as sharing support documents with customers or forms with external applicants who aren’t part of a trusted domain.

How flow works:

When a user needs to share a file publicly, they submit a request by uploading the file through a form. The admin reviews the request and decides whether to approve or reject it.

- If approved, an anonymous sharing link is generated and shared with the user. 
- If rejected, the uploaded file is automatically <a name="_int_ehugy65b"></a>deleted and a notification is sent to the user.

This approach helps organizations control external sharing by adding visibility, governance, and approval-based access.
